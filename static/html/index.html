Home
HTTP Method Path Description
GET / Fetches home page

Users
HTTP Method Path Description
GET /users Fetches all Users data from the database
GET /users/:id Fetches a single User's data from the database with the id provided
POST /users/register Adds a new user to the database.
PATCH /users/update/:id Updates a single user with the id value provided.
DELETE /users/delete/:id Deletes a single user's data from the database with the id provided

Cars
HTTP Method Path Description
GET /cars Fetches all cars from the database
GET /cars/:id Fetches a single car's data from the database with the id provided.
GET /cars/newadditions Fetches the 4 latest cars added to the database from the most recent.
POST /cars/addcar Adds a new car to the database.
PATCH /cars/update/:id Updates a single car with the id value provided.
DELETE /cars/delete/:id Deletes a single car's data from the database with the id provided
